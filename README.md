# irec - Image RECovery service

Данный сервис используется для восстановления изображения datamatrix кода. Он не действует как самостоятельный считыватель кода, а только улучшает качество изображения, чтобы сервис распознавания мог извлечь данные из кода.

Сервис не выполняет локализацию. Планируется, что на его вход будет поступать локализованное изображение datamatrix. При помощи нейросети сервис обрабатывает изображение и выдаёт на выходе чёрно-белый datamatrix, который будет отправлен на распознавание.
